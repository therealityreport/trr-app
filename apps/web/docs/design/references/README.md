# Design References

Use this folder for local-only reference inputs that support frontend generation and review.

## Keep Here

- Captured screenshots from the current product
- Approved mood boards
- Annotated comparison images
- Exported review screenshots from Playwright

## Do Not Keep Here

- Hotlinked external image URLs
- Unlicensed stock assets with unclear provenance
- Temporary screenshots that are no longer part of a review decision

## Current Reference Direction

- TRR public/auth surfaces: editorial, atmospheric, brand-forward, sparse copy
- TRR admin surfaces: direct, operational, scannable, low ornament
- TRR games: playful but still brand-coherent, with motion used for feedback rather than spectacle
