# Monotype fonts (local copies)

This folder contains local copies of Monotype fonts used by the side menu for development and internal testing.

Included families:
- Plymouth Serial Regular (TTF)
- Plymouth Serial Bold (TTF)
- Gloucester OS MT Std Regular (OTF)

Note: Ensure your Monotype license permits self-hosting/copying within your repo. Avoid publishing these files publicly if your license does not allow redistribution.
